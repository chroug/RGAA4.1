RGAA 8.8 - Cas Négatifs (Erreurs Algo & IA)

La langue par défaut est le français.

1. Mot complet au lieu du code (Algo)

Welcome to the website.

2. Mauvais séparateur (Algo)

Le W3C exige un tiret, pas un underscore : Hello world

3. Code inventé ou mal tapé (Algo)

Hola amigo

4. Attribut vide (Algo)

Guten Tag

5. Anglais déclaré sur du Français (IA)

Ceci est une phrase purement en français, mais balisée comme de l'anglais.

6. Espagnol déclaré sur de l'Allemand (IA)

Ich bin ein Berliner und ich liebe Wurst.

7. Allemand déclaré sur de l'Anglais (IA)

The quick brown fox jumps over the lazy dog.
